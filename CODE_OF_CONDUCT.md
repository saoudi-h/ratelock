# Community Guidelines

RateLock is an open-source project built by developers, for developers. We keep things simple.

## Be Respectful

Treat everyone with respect. Disagreements happen — handle them constructively. No personal attacks, harassment, or trolling.

## Be Helpful

If you see a question you can answer, a bug you can reproduce, or documentation you can improve — go for it. The project grows when people contribute.

## Keep It Professional

This is a workspace for building software. Keep discussions focused on the project. Spam, self-promotion, and off-topic content will be removed.

## No Tolerance for Abuse

Repeated violations of these guidelines will result in being blocked from the repository and related community spaces.

That's it. Build cool things.
